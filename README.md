## **<span style="color:green;">Estrutura do projeto</span>**
```
AQUOS/
├── 📁 backend/                
│   ├── 📁 src/
│   │   ├── 📁 config/         # Configurações do sistema
│   │   ├── 📁 controllers/    # Lógica de rotas (endpoints)
│   │   ├── 📁 models/         # Modelos de dados
│   │   ├── 📁 schemas/        # Validação de dados 
│   │   ├── 📁 services/       # Regras de negócio 
│   │   ├── 📁 utils/          # Funções auxiliares
│   │   ├── 📄 __init__.py
│   │   └── 📄 main.py         # Arquivo principal
│   ├── 📁 tests/              # Testes automatizados
│   ├── 📁 venv/               # Ambiente virtual (Ignorado no Git)
│   └── 📄 requirements.txt    # Bibliotecas Python
├── 📁 frontend/               # Interface React (Vite)
│   ├── 📁 node_modules/       # Dependências Node (Ignorado no Git)
│   ├── 📁 public/             # Ativos estáticos públicos
│   ├── 📁 src/
│   │   ├── 📁 assets/         # Imagens e estilos
│   │   ├── 📄 App.jsx         # Componente principal
│   │   ├── 📄 index.css       # Estilos globais
│   │   └── 📄 main.jsx        # Ponto de entrada React
│   ├── 📄 .gitignore          # Ignore local do frontend
│   ├── 📄 eslint.config.js    # Linter de código
│   ├── 📄 index.html          # Página principal HTML
│   ├── 📄 package.json        # Dependências e scripts
│   └── 📄 vite.config.js      # Configurações do Vite
├── 📄 .env                    # Chaves privadas (Ignorado no Git)
├── 📄 .env.example            # Modelo para configuração local
├── 📄 .gitignore              # Ignore global da raiz
└── 📄 README.md               # Documentação do projeto
```
